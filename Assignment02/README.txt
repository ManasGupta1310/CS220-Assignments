The "Report.pdf" file contains detailed description for both the CLA as well as Johnson Counter.

The report includes motivation, theory, truth tables, circuit diagrams, advantages and disadvantages of both the circuits.


Testing:-

The testing of the code is very straight forward.
After extracting the zip files, run the verilog files through command line or any suitable IDE supporting verilog:-

iverilog -o <name of test bench file>.vvp <name of test bench file>.v

vvp <name of test bench file>.vvp


After successful execution of these files, output will be displayed in the terminal/command line.
